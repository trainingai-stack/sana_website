## 📋 嵌入视频的步骤

### 方案一：使用 B 站视频（推荐）

1. **在 B 站找到合适的视频**
   - 访问 https://www.bilibili.com
   - 搜索医院名称 + "环境" 或 "vlog"

2. **获取视频 ID**
   - 打开视频页面
   - 从 URL 中提取视频 ID（BV 号）
   - 例如：`https://www.bilibili.com/video/BV1xxxx` → `BV1xxxx`

3. **更新代码**
   打开 `/src/app/hospitals/page.tsx`，找到 `videoTours` 数组，更新视频信息：

   ```typescript
   const videoTours = [
     {
       hospital: { zh: '北京协和医院', en: 'Peking Union Medical College Hospital' },
       title: { zh: '百年协和新貌 - 门诊大楼全景展示', en: 'Century-Old PUMCH - Outpatient Building Tour' },
       platform: 'Bilibili',
       videoId: 'BV1实际视频 ID',  // 替换为真实 BV 号
       thumbnail: '/video-thumb-pumch.jpg',
       duration: '8:35',
       views: '12.5 万'
     },
     // ... 其他视频
   ]
   ```

### 方案二：使用 iframe 嵌入（需要进一步开发）

如果需要直接在页面中播放视频，可以添加视频播放组件。这需要：

1. 安装视频播放器库（如 `react-player`）
2. 创建视频播放弹窗或页面
3. 配置视频平台的嵌入权限

## 📸 自定义缩略图

如果要使用真实视频缩略图：

1. **下载缩略图**
   - 从视频平台下载或使用截图工具
   - 推荐尺寸：1280x720 (16:9)

2. **添加到项目**
   - 放入 `/public` 目录
   - 文件名示例：`video-thumb-pumch.jpg`

3. **更新代码**
   - 修改 `thumbnail` 字段为实际文件名

## 🔗 视频链接跳转

目前视频卡片设计为点击后跳转到视频平台。如果要实现跳转：

1. **添加视频 URL 字段**
   ```typescript
   videoUrl: 'https://www.bilibili.com/video/BV1xxxx'
   ```

2. **添加点击跳转处理**
   - 使用 `window.open(videoUrl, '_blank')`
   - 或使用 Next.js 的 `Link` 组件

## 💡 最佳实践建议

1. **视频选择标准**
   - 画质清晰（至少 720p）
   - 时长适中（5-15 分钟）
   - 内容真实展示医院环境
   - 观看量较高（代表受欢迎程度）

2. **版权注意**
   - 优先选择官方账号发布的视频
   - 避免商业用途侵权
   - 可在视频描述中注明来源

3. **更新频率**
   - 定期检查视频链接是否有效
   - 根据季节或医院变化更新视频
   - 收集用户反馈优化视频选择
